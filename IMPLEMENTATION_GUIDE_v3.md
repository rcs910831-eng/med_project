## 시스템 개요

### 🎯 목적
PHARMA-HYBRID v3.0은 의료 전문가가 처방을 검토하고 약물 안전성을 즉시 검증할 수 있는 **의료 수준의 의사결정 지원 시스템**입니다.

### ✨ 핵심 기능

| 기능 | 설명 | 임상 기준 |
|------|------|---------|
| **약물 정보 검색** | 32개 약물 완전 정보 제공 | FDA/NIH 최신 가이드라인 |
| **실시간 안전 검증** | 약물 상호작용, 금기, 중복 약물 검출 | Black Box Warning 포함 |
| **모니터링 일정** | 환자별 맞춤 검사 스케줄 | 임상 표준 기반 |
| **처방 분석** | 질환별 약물 효능 및 부작용 | 임상시험 증거 |

## 주요 개선사항

### v2.0 → v3.0 변경사항

#### 1. 약물 데이터베이스 확대
```
✅ 2개 약물 → 32개 약물 (16배 증가)

✅ 실제 임상시험 증거 및 FDA 기준 반영
```

#### 2. 안전 검증 강화
```
✅ 자리먹이 오류 제거 (데이터 구조 표준화)

✅ 중복 약물 클래스 검사 (14개 약물 클래스)

```
✅ 명확한 페이지 네비게이션 (5개 독립 페이지)
✅ 색상 코딩된 심각도 (빨강/주황/녹색)

#### 4. 임상 근거 강화

## 사용자 가이드

### 📱 페이지별 사용법

#### 1️⃣ **👤 환자 정보 입력**

**목적:** 환자 데이터 등록 및 처방약 입력

**단계:**
1. 환자 기본 정보 입력 (이름, 나이, 성별, 진료과)
2. 의료기관/의사 정보 입력
3. 주요 질환 추가
4. 처방약 추가

**예시 (P014 파킨슨병 환자):**
```
환자명: 김파킨
나이: 68
성별: 남성
진료과: 신경과
질환: 파킨슨병, 심부전
처방약: 마도파, 롬코르, 라식스, 잇테놀
```

---

#### 2️⃣ **💊 약물 정보 검색**

**목적:** 특정 약물의 완전한 임상 정보 조회

**검색 방식:**
- **약물명으로 검색:** 약물 선택 → 전체 정보 표시
- **질환으로 검색:** 질환 입력 → 해당 약물 목록

**표시 정보:**
```
├─ 기본 정보 (성분명, 분류, 질환)
├─ 용량 (단일, 용법, 형태, 월량)
├─ 적응증 (임상적 사용)
├─ 임상 증거 (효능, 임상시험)
├─ 경고 (🔴 심각, ⚠️ 주의)
├─ 상호작용 (다른 약물과의 반응)
├─ 부작용 (발생률 포함)
├─ 모니터링 (필요한 검사 및 빈도)
└─ 가격 (개당/월액/보험)
```

**예시 (마도파 검색 결과):**
```
💊 마도파
성분명: Levodopa/Benserazide
분류: 파킨슨병 치료제
용량: 125mg 1일 3회

🔴 경고:
- 중단 절대 금지 (악성 신경이완제 증후군)
- 식사와 분리 섭취 필수
- MAOI 절대 금기
```

---

#### 3️⃣ **⚠️ 안전성 검증**

**목적:** 현재 처방약의 안전성을 종합적으로 평가

**검증 항목:**

| 항목 | 의미 | 예시 |
|------|------|------|
| **안전 점수** | 0-100 (높을수록 안전) | 85/100 안전 |
| **심각한 문제** | 치명적 상호작용 수 | 0개 (안전) |
| **경고** | 주의가 필요한 상호작용 | 2개 (모니터링) |
| **중복 약물** | 같은 클래스 약물 | 0개 (적절) |

**검증 결과:**

✅ **안전 (70점 이상)**
```
처방 가능, 정기 모니터링 권장
```

⚠️ **주의 (50-69점)**
```
모니터링 강화 필수, 필요시 약물 조정
```

🔴 **위험 (30-49점)**
```
약물 조정 권장, 의료진 협의
```

🔴 **매우 위험 (<30점)**
```
즉시 약물 변경 필수
```

**예시 - P014 위험한 처방:**
```
🔴 치명적 상호작용:
  리텔(리튬) + 잇테놀(ACEi)
  → 고칼륨혈증 위험
  → 조치: 즉시 제거

🟡 중복 약물:
  베타차단제: 롬코르 1개 ✅

⚠️ 금기약물:
  심부전 환자에게 NSAIDs 금지
```

---

#### 4️⃣ **📊 처방 분석**

**목적:** 처방의 임상적 적합성과 통계 분석

**분석 내용:**
- 질환별 약물 구성
- 약물 통계 (수, 분류)
- 상호작용 매트릭스
- 중복 약물 확인

**예시 - P012 크론병 처방:**
```
질환: 크론병

💊 처방약:
├─ 휴미라 (40mg 2주 1회) - TNF억제제
│  └─ 효능: 관해율 60-70%
├─ 이뮤란 (50mg 1일 1회) - 면역억제제
│  └─ 경고: CBC 월 1회 필수
├─ 펜탁사 (1g 1일 3회) - 5-ASA
│  └─ 효능: 관해율 50-60%
└─ 엽산 (5mg) - 보조약

🔗 상호작용 분석:
✅ 휴미라 + 이뮤란: 병용 권장 (효과 증강)
⚠️ 이뮤란 + 아자티오프린: 절대 금기 ❌
```

## 🎓 의료 전문가용 팁

### ✅ 시스템 활용 시 best practice

1. **처방 입력 전 검증**
   - 모든 처방약을 먼저 입력
   - 안전성 검증 실행
   - 문제점 확인 후 수정

2. **약물 정보 활용**
   - 모니터링 항목을 환자 기록에 추가
   - 경고사항을 환자 교육에 활용
   - 부작용을 사전에 설명
