## 📋 지적된 주요 문제점 정리

### 1️⃣ **UI/UX 레이아웃 문제**
- ❌ 위에 흰 바탕 (불필요)
- ❌ 고객선택 2개 중복
- ❌ 오른쪽에 다른 처방전이 섞여 있음
- ❌ 중앙과 오른쪽 정보 일관성 부족

### 2️⃣ **로직 모순**
- ❌ "실시간 모니터링 15%" vs "충돌 없음 88%" → 모순
- ❌ 중앙 약물 내용 ≠ 오른쪽 약물 내용
- ❌ 이것도 충돌인데 감지 안 함

### 3️⃣ **처방전 인식 부족**
- ❌ 처방전 날짜 감지 안 함
- ❌ 약물별 1회 투여량 없음
- ❌ 1일 투여횟수 없음
- ❌ 총 투약 일수 없음
- ❌ 각 알약 주의사항 없음

### 4️⃣ **질병 우선순위 혼재**
- ❌ 1순위/2순위 표시 없음
- ❌ 어떤 약물이 어느 병의 약물인지 불명확
- ❌ 약물-질환 매핑 없음

### 5️⃣ **표 구성 미흡**
- ❌ 복용약물 표와 약물 상세정보 표 분리 안 됨
- ❌ 약물 효능-약물명 매핑 없음
- ❌ 약물 이미지(알약 생김새) 없음
- ❌ 알약별 주의사항 별도 표시 없음

### 6️⃣ **임상 정보 미흡**
- ❌ 약물 효과 시간 변화 추적 안 함
- ❌ 언제는 잘 듣고 언제는 안 듣는 패턴 감지 안 함
- ❌ 다병원 중복 확인은 표시만 하고 활용 안 함

```python
약물이미지 = {
    "타세바정": {
        "약물명": "Tarceva",
        "성분명": "Erlotinib 150mg",
        "이미지URL": "pharma_db/erlotinib_150mg.png",
        "규격": "황갈색 타원형",
        "각인": "T 150",
        "크기": "약 12mm",
        "보관": "실온, 밀폐용기"
    }
}
```

#### 3-2. 약물-질환 매핑표
```
약물명          | 질병          | 질병순위 | 효능       | 근거
────────────────────────────────────────────────────
타세바정        | 폐암(EGFR+)   | 1순위    | 83% 반응   | OPTIMAL
무코스타정      | 위염          | 2순위    | 완화       | 표준
베아제정(제거)  | -             | -        | 불적절    | 금기
```

#### 3-3. 알약별 주의사항 매핑
```
약물 → 주의사항 구분
- 간독성 주의 약물: AST/ALT 월 1회
- 피부 부작용: 발진 시 즉시 보고
- 폐독성 주의: 기침 감시
```
